●Quick Setup Guide

Get FleetOS running
in minutes

Follow these steps to deploy the full stack — Traccar backend, PostgreSQL, web UI, and mobile app.

⚡Docker path:~10 min
🔧Manual path:~30 min
📱Mobile app:~20 min extra
Setup progress 0 of 6 complete
1
Prerequisites
Install Docker and Git on your server
2 min
▼
💡
Docker is the fastest path — it handles Traccar, PostgreSQL, pgAdmin, and the web UI in one command. If you prefer manual installation, jump to Step 3.
a
Install Docker + Docker Compose
Ubuntu / Debian:
bash
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER
newgrp docker
docker compose version  # should print v2.x.x
macOS / Windows: Install Docker Desktop from docker.com — includes Compose.
b
Download FleetOS files
bash
# Extract the downloaded files into a folder
mkdir fleetos && cd fleetos
# Copy in: docker-compose.yml, config/, database/, web/, nginx.conf
Your folder should look like this:
fleetos/
  ├── docker-compose.yml ← main launcher
  ├── nginx.conf
  ├── config/
  │   └── traccar.xml ← edit password here
  ├── database/
  │   └── schema.sql
  └── web/
      └── index.html ← web UI
2
Configure passwords
Set your database password before launching
2 min
▼
⚠️
Do this before running Docker. The password must match in both files or Traccar won't connect to Postgres.
a
Create the .env file
Run in your fleetos/ folder:
bash
echo "POSTGRES_PASSWORD=YourSecurePass123" > .env
b
Update config/traccar.xml
Open config/traccar.xml and find this line — replace with the same password:
xml
<entry key='database.password'>YourSecurePass123</entry>
c
Optional: configure email notifications
Also in traccar.xml, update the SMTP entries for alert emails. You can skip this and do it later.
3
Launch the stack
One command starts everything
3 min
▼
a
Start all services
bash
cd fleetos
docker compose up -d

# Watch the logs until you see Traccar start:
docker compose logs -f traccar
Wait for the line: Main: server is running — takes ~30 seconds on first run.
b
Verify all containers are healthy
bash
docker compose ps
All 4 containers should show Up or healthy:
fleetos-postgres
:5432
Database server
fleetos-traccar
:8082
GPS backend + API
fleetos-web
:3000
FleetOS web UI
fleetos-pgadmin
:5050
Database manager
4
Create your admin account
First-time Traccar setup + connect FleetOS
2 min
▼
✅
Traccar creates the first account as admin automatically — no separate setup needed.
a
Create your Traccar admin account
Open http://localhost:8082 in your browser. You'll see Traccar's default web interface. Click Register and create an account — the first account is automatically made an admin.
b
Open FleetOS and log in
Now open http://localhost:3000 — this is the FleetOS UI. Use the same credentials you just created.
login fields
Server URL : http://localhost:8082
Email      : your@email.com
Password   : yourpassword
c
Or use a demo server to test immediately
Click "Use Demo Server" on the login screen to connect to Traccar's public demo with pre-loaded vehicles — no setup required.
demo credentials
Server URL : https://demo.traccar.org
Email      : demo@traccar.org
Password   : demo
5
Connect your GPS devices
Register devices and configure trackers to send data
5 min
▼
a
Add a device in Traccar
In the FleetOS Devices tab, or directly at http://localhost:8082 → Devices → Add. You need the device's IMEI number (printed on the tracker or in its settings).
fields
Name      : Alpha-01  (any name you want)
Unique ID : 123456789012345  (IMEI number)
b
Configure your GPS tracker
Send an SMS to your tracker (or use its config software) to point it at your server. Common SMS commands:
GT06 / Concox
SERVER,0,YOUR_SERVER_IP,5023,0#
Sinotrack ST-901
804admin YOUR_SERVER_IP 5036
Teltonika (via Configurator app)
GPRS → Server → IP: YOUR_SERVER_IP  Port: 5027
c
Port reference — which port for your tracker
Device / ProtocolPortProtocol
GT06, Concox, Jointech5023GT06
Teltonika FMB, FMC, FMP5027Teltonika
Queclink GV300/500/6005004GL200
Sinotrack ST-901/9065036Sinotrack
Meitrack T399 / MT905020Meitrack
Traccar Client (mobile)5055OsmAnd
Generic NMEA over TCP10000NMEA
💡
Don't see your device? Traccar supports 200+ protocols. Check traccar.org/devices for the full list and port number.
d
Track using Traccar Client (phone as a tracker)
Install Traccar Client from the App Store / Play Store. Open it, set Server URL to http://YOUR_IP:5055 and Device Identifier to the IMEI you registered. Tap Start — your phone appears on the map instantly.
6
Mobile app setup
Build and run the React Native app on iOS / Android
20 min
▼
⚠️
Requires Node.js 18+, and Xcode (iOS) or Android Studio (Android) installed. Skip this step if you only need the web UI.
a
Bootstrap the React Native project
bash
npx react-native@latest init FleetOSMobile \
  --template react-native-template-typescript
cd FleetOSMobile
b
Install dependencies
bash
npm install \
  @react-navigation/native \
  @react-navigation/bottom-tabs \
  @react-navigation/stack \
  react-native-maps \
  @react-native-async-storage/async-storage \
  react-native-safe-area-context \
  react-native-screens
c
Copy the app file + configure Google Maps (Android)
bash
# Copy App.tsx from the fleetos/mobile/ folder
cp ../fleetos/mobile/App.tsx ./App.tsx
For Google Maps on Android, add your API key in android/app/src/main/AndroidManifest.xml:
xml
<meta-data
  android:name="com.google.android.geo.API_KEY"
  android:value="YOUR_GOOGLE_MAPS_API_KEY"/>
💡
Get a free Google Maps API key at console.cloud.google.com → APIs → Maps SDK for Android. iOS uses Apple Maps by default (no key needed).
d
Run on device
iOS
cd ios && pod install && cd ..
npx react-native run-ios
Android
npx react-native run-android
On the login screen, enter your server IP (not localhost — use your machine's actual IP address like http://192.168.1.x:8082).
🚀
FleetOS is live!

Your full-stack GPS fleet platform is running. Open these URLs to get started:

Traccar API / Default UI

pgAdmin (DB Manager)
